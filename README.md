# 🤖 ZIPO – Warehouse Package Navigator

## Project Idea

**ZIPO** is a smart warehouse robot designed to automate package movement using grid-based navigation.
The robot can move between locations, calculate the shortest path, and detect obstacles in real time.
It connects to a control system through Wi-Fi, allowing users to send movement commands remotely.

This project demonstrates how robotics and embedded systems can be used to improve efficiency and safety in warehouse environments.

---

## Key Features

* Grid-based navigation system(Unity based)
* Shortest path calculation
* Real-time obstacle detection using sensors
* Remote control via Wi-Fi(Firebase)
* Autonomous movement between locations

---

## Technologies Used

* ESP32 Microcontroller
* Ultrasonic Sensor
* DC Motors and Servo Motor
* Wi-Fi Communication
* Pathfinding Algorithms
* Embedded Programming (Arduino / C)
